== Installation ==
1. Go to Plugins > Add New.
2. Search for \"webiots testimonials\".
1. Click \"Install Now\".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin. Look for \"Testimonials\" in the admin menu.

= What are the shortcodes? =

[webiots-ts] - To display your testimonials as a list or a grid or video. There are no complicated shortcode options.

`[webiots-ts]` => default layout
`[webiots-ts style=\"grid1\"]`
`[webiots-ts style=\"grid2\"]`
`[webiots-ts style=\"list1\"]`
`[webiots-ts style=\"list2\"]`
`[webiots-ts style=\"video\"]`
